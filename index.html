Utilisation : Utilisez la barre de défilement en bas pour rejouer les positions du véhicule.

Paramètres du fichier :

  • Accélération (acceleration)
  • Angle des roues (steering_angle)
  • Position essieu avant
  • Position centre
  • Position essieu arrière
  • Distance point de contrôle / trajectoire (control_point_distance_to_path)

Paramètres recalculés :

  • Distance essieu avant / trajectoire (front_axle_distance_to_path)
  • Distance centre / trajectoire (wheelbase_distance_to_path)
  • Distance essieu arrière / trajectoire (rear_axle_distance_to_path)
